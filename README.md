# Autohold

### FLEXCOMM0: UART Debug
- UART0_TXD: PORT0_PIN29
- UART0_RXD: PORT0_PIN30

### FLEXCOMM2: I2C INA219
- I2C2_SDA: PORT0_PIN26
- I2C2_SCL: PORT0_PIN27

### FLEXCOMM3: SPI MCP2515
- SPI3_MCLK: PORT0_PIN0
- SPI3_SO: PORT0_PIN2
- SPI3_SI: PORT0_PIN3
- SPI3_SSN:  PORT0_PIN4

### CTIMER0: PWM
- MATCH3: PORT1_PIN2
